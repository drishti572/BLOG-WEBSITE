Travellers Hub – Uttarakhand Travel Website 🌄

Travellers Hub is a responsive travel blog website that showcases beautiful tourist destinations of Uttarakhand. The website allows users to explore famous places, read travel blogs, view locations on a map, and share their experiences through comments.

The goal of this project is to create a simple and interactive travel website using HTML, CSS, and JavaScript.

Features ✨
1. Navigation Bar

Website logo

Destination search bar

Social media icons

Dark/Light theme toggle

2. Hero Section

Large banner showing Himalayan mountains

Introduction text for the travel website

3. Top Destinations

The website displays popular tourist places such as:

Kedarnath

Auli

Mussoorie

Nainital

Each destination card contains an image and a short description.

4. Map Section

An embedded Google Map helps users explore the location of Uttarakhand.

5. Blog Section

The blog section includes travel stories with videos about places like:

Kedarnath Temple Trek

Auli Snow Paradise

Mussoorie – Queen of Hills

Nainital Lake City

Tungnath Temple Trek

Rishikesh River Rafting

Each blog card contains:

Travel video

Date and description

Comment section

6. Comment System

Users can:

Enter their name

Write a comment

Post their experience under each blog.

7. Search Function

Users can search blog posts by destination name using the search bar.

8. Pagination

Only 3 blog posts are shown per page.
Users can move between pages using Previous and Next buttons.

9. Theme System

The website supports:

Light mode

Dark mode

Automatic theme based on time (sunrise, day, sunset, night)

Technologies Used 💻

HTML5 – Website structure

CSS3 – Styling and responsive design

JavaScript – Interactivity and functionality

Font Awesome – Icons

Google Fonts – Typography

Google Maps Embed – Location display

How to Run the Project ▶️

Download or clone the project.

Open the project folder.

Open index.html in any web browser.

The website will run locally without any server.
